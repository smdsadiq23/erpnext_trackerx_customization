How to Use AQL Tables

  1. Select Inspection Regime: Choose Normal, Tightened, or Reduced based on supplier history
  2. Choose Inspection Level: Select appropriate level based on product criticality
  3. Find Lot Size Range: Locate your lot quantity in the table
  4. Get Sample Size: Use the corresponding sample code and size
  5. Apply AQL Criteria: Accept lot if defects ≤ Acceptance Number, reject if defects ≥ Rejection Number
Note: This system implements ISO 2859-1 (MIL-STD-105E) standards for acceptance sampling by attributes.
